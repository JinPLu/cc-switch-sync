# CC Switch Remote 使用说明

把本机 Claude Code / Codex 的 Provider 配置同步到远程 Linux 服务器，并提供 SSH 登录和历史下载。

## 下载后点哪个

解压 zip 后，根目录只需要看这几个文件：

- Windows 安装：双击 `安装 Windows.bat`
- macOS 安装：双击 `安装 macOS.command`（如果被拦截，右键 → 打开）
- Windows 直接打开：双击 `打开 CC Switch Remote.bat`
- `src/` 是内部程序目录，普通使用不用打开

安装完成后会自动创建桌面快捷方式 `CC Switch Remote`，并打开首次设置向导。

## 安装实际做什么

安装只负责把工具装好，不会自动替你配置服务器。

Windows 安装会：

1. 检查 Git Bash 是否存在。
2. 把工具复制/更新到 `%USERPROFILE%\.cc-remote`。
3. 保留你的 `config.ini`、`servers.conf`、`history-downloads`。
4. 创建桌面和开始菜单快捷方式。
5. 打开首次设置向导。

macOS / Linux 安装会：

1. 检查 `ssh`、`scp`、`tar`、`python3`。
2. 把工具复制/更新到 `~/.cc-remote`。
3. 创建 `~/.local/bin/cc-remote` 命令链接。
4. 尝试创建桌面启动文件。
5. 打开首次设置向导。

## 首次设置需要准备什么

首次设置会引导你添加服务器、测试 SSH、确认远端初始化和同步配置。你需要提前准备：

- 服务器 Host / IP
- SSH 端口（通常是 22）
- SSH 用户名
- 可用的 SSH 密钥或密码
- 远端工作目录（可用 `~`）
- 如有需要，准备代理地址

远端初始化只会在你确认后执行，并且只更新远端 `~/.bashrc` 中的 `cc-switch-remote-kit` 标记块，会先备份原文件。

## 命令行使用

安装后可以运行：

```sh
cc-remote setup          # 首次设置向导
cc-remote                # 交互菜单
cc-remote doctor         # 检查依赖、插件、服务器
cc-remote add            # 添加服务器
cc-remote test [name]    # 测试 SSH
cc-remote sync [name]    # 同步配置
cc-remote connect [name] # SSH 登录
cc-remote history [name] # 下载会话历史
```

开发者一行安装：

```sh
curl -fsSL https://raw.githubusercontent.com/farion1231/cc-switch-sync/main/src/install.sh | bash
```

```powershell
irm https://raw.githubusercontent.com/farion1231/cc-switch-sync/main/src/install.ps1 | iex
```

## 安全边界

- 用户配置默认保存在 `~/.cc-remote/config.ini` 和 `~/.cc-remote/servers.conf`。
- 下载历史保存到 `~/.cc-remote/history-downloads/<server>/<timestamp>`。
- 同步 Claude 时只合并 `~/.claude/settings.json` 的 `env` 段，其他字段不动。
- Codex `auth.json` 默认需要确认才上传（`sync_codex_auth=confirm`）。
- 本地 auth/config/settings/sqlite 文件不会被历史下载覆盖。
